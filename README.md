# AI Firework Designer

An interactive AI-powered web app that turns text prompts into animated firework explosions using canvas particles and a Supabase Edge Function backend powered by Groq LLM.

Type something like *“red and blue patriotic fireworks”* and watch it explode into a custom visual firework show in real time.

## Demo

https://stephanie-murrile.github.io/Fourth-of-July-AI-Firework-Creator

Try prompts like:
- "golden sparkling fireworks over ocean"
- "red and blue patriotic fireworks"
- "neon cyberpunk explosion sky"
- "soft pastel fireworks for celebration"

The AI interprets your text and generates a unique animated firework pattern on the canvas in real time.

## Features

- AI-generated firework colors from text prompts
- Real-time canvas particle explosion system
- Fast backend using Supabase Edge Functions
- Groq LLM integration for prompt understanding
- Works in browser
- Custom explosion size and style per prompt

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Canvas API
- Supabase Edge Functions
- Groq API
